Detailed Description

template<typename T>
class Iterator< T >

Abstract iterator interface for traversing collections.

This template class defines the interface for the Iterator pattern. It provides pure virtual methods that concrete iterators must implement to traverse different types of collections.

Responsibilities:

  • Defines the interface for collection traversal
  • Provides abstract methods for navigation and access
  • Supports different element types through templating

TType of elements to iterate over
