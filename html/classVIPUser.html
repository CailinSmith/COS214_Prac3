Detailed Description

Concrete decorator that adds VIP status and priority messaging to users.

This decorator extends user functionality by adding a gold-colored VIP tag and provides priority messaging behavior. VIP users get special treatment when sending messages with a priority announcement.

Responsibilities:

  • Adds gold "(VIP)" tag to user names
  • Provides priority message sending with special announcement
  • Maintains all other AbstractUser functionality through delegation
