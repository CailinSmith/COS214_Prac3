Detailed Description

Concrete decorator that adds a green "Dog Whisperer" tag to user names.

This decorator extends user functionality by adding a green-colored role tag that appears when the user's name is displayed. Inspired by Discord-style roles.

Responsibilities:

  • Adds green "(Dog Whisperer)" tag to user names
  • Maintains all AbstractUser functionality through delegation
  • Provides custom message receiving behavior with role display
