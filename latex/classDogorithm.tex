\doxysubsection{Detailed Description}
Concrete chat room implementation for Dogorithm-\/themed discussions. 

This class extends \doxylink{classChatRoom}{Chat\+Room} to provide a specific themed chat room. It inherits all mediator functionality from \doxylink{classChatRoom}{Chat\+Room} and provides its own identification through the print method.

Responsibilities\+:
\begin{DoxyItemize}
\item Inherits all \doxylink{classChatRoom}{Chat\+Room} mediator functionality
\item Provides specific room identification as "{}\+Dogorithm"{}
\item Serves as concrete mediator for Dogorithm-\/themed conversations 
\end{DoxyItemize}
