\doxysubsection{Detailed Description}
Concrete decorator that adds a green "{}\+Dog Whisperer"{} tag to user names. 

This decorator extends user functionality by adding a green-\/colored role tag that appears when the user\textquotesingle{}s name is displayed. Inspired by Discord-\/style roles.

Responsibilities\+:
\begin{DoxyItemize}
\item Adds green "{}(\+Dog Whisperer)"{} tag to user names
\item Maintains all \doxylink{classAbstractUser}{Abstract\+User} functionality through delegation
\item Provides custom message receiving behavior with role display 
\end{DoxyItemize}
