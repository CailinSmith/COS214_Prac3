\doxysubsection{Detailed Description}
Concrete implementation of \doxylink{classAbstractUser}{Abstract\+User} that manages chat room memberships and command execution. 

This class serves as the Concrete Colleague in the Mediator pattern. It implements all abstract methods from \doxylink{classAbstractUser}{Abstract\+User} and uses the \doxylink{classCommand}{Command} pattern for deferred execution of messaging operations.

Attributes\+:
\begin{DoxyItemize}
\item chat\+Rooms\+: Vector of chat rooms the user is a member of
\item name\+: String identifier for the user
\item command\+Queue\+: Vector of commands awaiting execution
\end{DoxyItemize}

Responsibilities\+:
\begin{DoxyItemize}
\item Manages membership in multiple chat rooms
\item Implements message sending using \doxylink{classSendMessageCommand}{Send\+Message\+Command} and \doxylink{classLogMessageCommand}{Log\+Message\+Command}
\item Implements broadcasting using \doxylink{classBroadcastCommand}{Broadcast\+Command}
\item Maintains and executes a queue of commands
\item Provides access to chat room memberships 
\end{DoxyItemize}
