\doxysubsection{Detailed Description}
Abstract base class implementing the \doxylink{classCommand}{Command} pattern for deferred execution. 

This class serves as the \doxylink{classCommand}{Command} interface in the \doxylink{classCommand}{Command} pattern, allowing for encapsulation of messaging operations that can be queued and executed later. All concrete command implementations inherit from this class.

Attributes\+:
\begin{DoxyItemize}
\item room\+: Pointer to the chat room where the command should be executed
\item message\+: The message content associated with the command
\item from\+User\+: Pointer to the user who initiated the command
\end{DoxyItemize}

Responsibilities\+:
\begin{DoxyItemize}
\item Stores command parameters for deferred execution
\item Defines interface for command execution
\item Provides base functionality for concrete command implementations
\item Enables undo/redo functionality and command queuing 
\end{DoxyItemize}
