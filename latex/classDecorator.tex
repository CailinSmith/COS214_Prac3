\doxysubsection{Detailed Description}
Base decorator for dynamically adding responsibilities to \doxylink{classAbstractUser}{Abstract\+User} objects. 

Attributes\+:
\begin{DoxyItemize}
\item user\+: Pointer to the wrapped \doxylink{classAbstractUser}{Abstract\+User}.
\item owns\+User\+: Indicates ownership for memory management.
\end{DoxyItemize}

Responsibilities\+:
\begin{DoxyItemize}
\item Delegates all \doxylink{classAbstractUser}{Abstract\+User} methods to the wrapped user.
\item Allows runtime extension of user behavior via derived decorators. 
\end{DoxyItemize}
